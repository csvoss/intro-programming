Sikuli Script

Sikuli Script is a tool that allows you to create programs which can recognize parts of your screen, click on buttons, and type in keyboard commands. Once you install it, here are some ideas for projects to work on:
  • Write a program to play QWOP for you!
  • Write a program to play 2048 (or its really big cousin, 9007199254740992) for you!
    • For added challenge: Can you write a program which stores the 2048 game’s board state in a Python object?
  • Write a program to play Cookie Clicker for you. In particular, see if you can recognize when to start buying the power-ups from the store.
